# 第13章：教程自动更新与仓库维护

> 本章介绍如何设置自动化流程，让教程能够持续更新和维护。

---

## 13.1 自动化维护策略

### 目标
- 定期检查内容过期情况
- 自动更新版本号和命令示例
- 保持教程与 OpenClaw 最新版本同步

### 实现方式
使用 `complex-task-automator` Skill 配置自动化工作流：

```bash
python scripts/task-run.py workflows/openclaw-tutorial-auto/workflow-full.yaml
```

---

## 13.2 Git 工作流

### 自动提交
教程更新后自动 Git commit 和 push：

```bash
git add .
git commit -m "auto: update chapter N"
git push origin main
```

### 分支策略
- `main`: 稳定版本
- `draft`: 草稿分支
- `feature/*`: 新章节分支
