# ⚡ 快速开始

> 🕐 5 分钟从零搭建 OpenClaw Agent 并运行第一个自动化任务

---

## 📋 前置要求

| 项目 | 要求 | 检查命令 |
|------|------|---------|
| 操作系统 | Linux / macOS / Windows (WSL2) | `uname -a` |
| Node.js | 18+ | `node --version` |
| 网络 | 可访问 GitHub | `curl -s https://github.com` |

---

## 🔨 安装步骤

### 1️⃣ 安装 OpenClaw

```bash
curl -fsSL https://openclaw.dev/install.sh | bash
```

✅ **验证安装：**
```bash
openclaw --version
# 预期输出: openclaw 2026.x.x
```

---

### 2️⃣ 初始化配置

```bash
openclaw init
```

> 📁 这会创建 `~/.openclaw/` 目录并生成默认配置文件。

---

### 3️⃣ 启动 Gateway

```bash
openclaw start
```

> [!TIP]
> ⏳ 首次启动会自动配置设备身份和网络，可能需要 10-30 秒。

✅ **验证启动成功：**
```bash
openclaw status
# 预期输出: Gateway running on port 3000
```

---

### 4️⃣ 安装一个 Skill

```bash
# 从 ClawdHub 安装社区 Skill
openclaw skill install web-searcher
```

✅ **验证安装：**
```bash
openclaw skill list
```

---

### 5️⃣ 运行自动化任务

```bash
# 发送一条消息测试
openclaw message send --channel feishu --target <your-chat-id> --message "Hello from OpenClaw!"

# 创建定时任务
openclaw cron add --name "daily-report" --schedule "0 9 * * *" --command "openclaw run generate-report"
```

---

## ✅ 安装完成检查清单

- [ ] `openclaw --version` 输出正常
- [ ] `~/.openclaw/` 目录已创建
- [ ] `openclaw status` 显示 Gateway 运行中
- [ ] 至少安装了一个 Skill
