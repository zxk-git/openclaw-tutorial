## 🔧 核心功能

### 1. 全链路文档工作流

将复杂文档项目拆分为：
- **信息搜集**：通过 Tavily/DDG 搜索最新资料
- **大纲管理**：生成和管理章节大纲
- **内容撰写**：基于搜索结果撰写高质量内容
- **Git 自动推送**：commit + push 到远程仓库

### 2. 六维度质量检测

| 维度 | 说明 | 权重 |
|------|------|------|
| 内容充实度 | 信息深度和广度 | 20% |
| 结构完整性 | 章节组织和逻辑 | 20% |
| 代码质量 | 代码示例准确性 | 15% |
| 可读性 | 排版和易读性 | 15% |
| 教学价值 | 学习效果 | 15% |
| 时效性 | 信息是否最新 | 15% |

### 3. Cron 自动调度

通过 OpenClaw Cron 系统实现 24/7 自动执行：

```json
{
  "schedule": "0 */4 * * *",
  "task": "搜索最新信息并优化已有章节"
}
```

### 4. 任务拆解与管理

- 复杂任务自动拆分为子任务
- 依赖关系管理
- 并行执行能力
- 失败重试机制
- 断点恢复支持

## 💡 使用方式

### DRY_RUN 模式（安全预览）

在执行前先预览将要进行的操作，不做任何实际变更。

### 典型工作流

1. 定义项目大纲（章节列表）
2. 配置信息源（搜索引擎和关键词）
3. 启动自动生成
4. 质量检测和人工审核
5. Git 自动提交推送
