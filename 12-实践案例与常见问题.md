# 第12章：实践案例与常见问题

> 本章汇总 OpenClaw 的实践案例和常见问题解答。

---

## 12.1 案例一：个人知识助手

### 场景
构建个人知识管理助手，自动整理和搜索资料。

### 使用的 Skills
- `tavily-search`: 网络搜索
- `memory`: 知识存储
- `summarize`: 内容摘要

### 配置要点
1. 配置 Tavily API Key
2. 设置记忆分类体系
3. 配置每日归档 Cron 任务

---

## 12.2 案例二：项目监控机器人

### 场景
监控 GitHub 项目状态，自动推送到飞书。

### 实现方法
1. 安装 `github` Skill
2. 配置飞书通道
3. 设置 Cron 定时检查
4. 自动推送 PR/Issue 状态

## 12.3 常见问题汇总

### Q: OpenClaw 占用多少资源？
A: Gateway 常驻内存约 100-200MB，Agent 按需启动。

### Q: 支持哪些 AI 模型？
A: 支持 GPT-4.1、Claude、Gemini 等，通过配置切换。

### Q: 数据存在哪里？
A: 所有数据存在 `~/.openclaw/` 目录下，完全本地化。
